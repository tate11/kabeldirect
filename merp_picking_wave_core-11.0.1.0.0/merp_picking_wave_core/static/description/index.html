mERP Picking Wave Core module

The module adds specific information about related warehouses and locations on a picking wave form. And validates types of all pickings added into the wave, thus it is not possible to add pickings of different types into one picking wave.

It also adds On Hold state for picking waves (used in mERP Picking Wave module if not all pickings are confirmed yet and appropriate behaviour is selected).

Notes

This module is designed to work only with "Ventor" application that can be downloaded from:

  1. https://merpapp.com/ website - version obtained from here is suitable for all types of organizations
  2. Google Play - has limitations, not possible to customize app by request

"Ventor" app work both on regular mobile phones with Android 4.1+ and with professional barcode scanners. For full list of supported devices check F.A.Q.
