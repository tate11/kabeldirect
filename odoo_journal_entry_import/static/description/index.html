Journal Entry Import from Excel

This module allow you to import journal items from excel file.

Following fields can be import:

  • ACCOUNT CODE
  • PARTNER
  • LABEL
  • AMOUNT CURRENCY
  • CURRENCY
  • DEBIT
  • CREDIT
  • DUE DATE
  • REFERENCE
  • DATE

Make sure you have xlrd Python module installed. https://pypi.python.org/pypi/xlrd

For more details please see Video in Live Preview section. Excel sample is available inside module.
